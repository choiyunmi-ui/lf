UIO 가이드
  • 1. card_template_wrap은 card_header,card_template,card_footer를 감쌀때만 사용한다.
  • 2. 카드사이의 여백이 필요없을 경우 클래스명 margin_none 추가
  • 3. 서브페이지(기본구조)

    <div id="wrap"> <div class="top_fix"></div> -> 상단에 location이나 filter가 고정되야할때 사용한다.(header는 제외) <div id="container" role="main"> <div id="content"> 컨텐츠 </div> </div> <div id="footer" role="contentinfo">푸터</div> <div class="bottom_nav">하단 메뉴</div> </div>

    * 서브페이지 상단에 고정되는 UIO에 따라 wrap에 각 클래스명 선언
    - location 고정 : sub
    - header 고정 : sub2
    - location + filter 고정 : sub3
    - 그 이외의 경우 wrap에 style값으로 padding-top 적용
    * header와 다른 UIO가 고정으로 적용되어야 할 경우
    - header를 top_fix 내에 넣어준다. / 해당 경우 이외에는 header 단독 사용
